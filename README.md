# EV Vehicle Charge Demand Forecasting

## Project Overview

This project predicts Electric Vehicle (EV) charging demand using Machine Learning techniques. The model analyzes historical EV data and forecasts future charging requirements.

## Features

* EV demand forecasting
* Data preprocessing
* Machine Learning model training
* Future demand prediction
* Data visualization

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Jupyter Notebook

## Dataset

The project uses Electric Vehicle population and charging-related datasets for training and prediction.

## Files Included

* EV_Adoption_Forecasting.ipynb
* web.py
* Electric_Vehicle_Population_By_County.csv
* forecasting_ev_model.pkl

## Applications

* EV charging station planning
* Energy demand forecasting
* Smart city infrastructure planning
